Descriptions

Description A: Fluid buildup behind tympanic membrane, often following URI
Description B: Excessive/properly formed cerumen causing blockage
Description C: Episodic vertigo, tinnitus, hearing loss, ear fullness
Description D: Subjective perception of sound without external source

Treatments

Treatment A: Antibiotics, pain management, possible myringotomy
Treatment B: Cerumenolytic drops, irrigation, manual removal
Treatment C: Diuretics, low-sodium diet, vestibular rehab
Treatment D: Sound therapy, hearing aids, counseling
Clinical Tip: Always use otoscope to visualize tympanic membrane before irrigation. Never irrigate if TM is perforated. Document color, position, and light reflex of TM.

Audiogram Interpretation Guide:

Normal Hearing: 0-25 dB HL at all frequencies

Mild Loss: 26-40 dB HL (difficulty with soft speech)

Moderate Loss: 41-70 dB HL (needs hearing aid)

Severe Loss: 71-90 dB HL (loud speech only)

Profound Loss: 91+ dB HL (may benefit from cochlear implant)
